Responsive Web Design

What does it mean?

  • Quick to react
  • Receptive
  • Flexible
  • Sensitive

Why is it important?

  • Many different devices that you can use to go online
  • Different OS's on this devices
  • You can resize the browser window

A web page should look good on any device

Responsive vs adaptive

Where we can find the examples of Responsive web design?

What we should to do, to create RWD?

  • Set the viewport
  • Do not use large fixed width elements
  • Use CSS media queries
  • Flexbox

Set the viewport

What is it?

This is a meta tag with the name viewport

Why?

Browser needs to know how to scale the page for your screen

Flex-grow

Sets how much of the available space in the flex container should be assigned to that item

Flex-shrink

Sets the flex shrink factor of a flex item.

Flex-basis

Sets the initial main size of a flex item

The flex-flow shorthand

We can combine the two properties flex-direction and flex-wrap into the flex-flow shorthand

flex-flow

							
								.box {
									display: flex;
									flex-flow: row wrap;
								}
